# Highnlow Brand Website

[https://highnlow.pingpong.house](https://hayhyang.github.com.io/highnlow)

## 개발 환경 및 사용 라이브러리

- npm
- parcel-bundler
- scss

## 폴더 구조 설명

- `assets`: static source
  - `css`
  - `images`
  - `js`
  - `videos`
- 'index.html'

## 설치

```shell
npm install
```

## Dev 환경 실행

```shell
npm run dev
```

[http://localhost:1234](http://localhost:1234) 로 열림
